######################
pyuplift documentation
######################

**pyuplift** is a scientific uplift modeling library. It implements variable selection and transformation approaches. pyuplift provides API for work with such an uplift datasets as `Hillstrom Email Marketing <https://blog.minethatdata.com/2008/05/best-answer-e-mail-analytics-challenge.html>`_ and `Criteo Uplift Prediction <http://ailab.criteo.com/criteo-uplift-prediction-dataset>`_.
